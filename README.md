# Matchtags

## Highlights matching html/xml tags in Neovim



#### Enable or Disable
```
let g:load_matchtags = 1 or 0
```

#### Define filetypes of highlights
```
  let g:matchtags_filetypes = {
        \ 'html' : 1,
        \ 'xhtml' : 1,
        \ 'xml' : 1,
        \ 'jinja' : 1,
        \ 'php' : 1,
        \ 'vue' : 1,
        \ 'svelte' : 1,
        \ 'javascriptreact' : 1,
        \ 'typescriptreact' : 1,
        \ 'eruby': 1,
        \ }
```
